LiveSpecs Examples

LiveSpecs is a library that allows you to write code samples that are be displayed on your documentation site, but also run as part of your test suite.

For documentation on how to use LiveSpecs, please visit the repo.

Here you can find examples on how to use LiveSpecs, both as an AMD (require.js) module, or as a global variable.
